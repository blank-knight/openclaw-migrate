# 技术栈

## 语言 & 运行时

| 技术 | 版本 | 用途 |
|------|------|------|
| Node.js | >= 18 | 运行时（与原项目一致） |
| JavaScript (ES2022) | - | 开发语言（与原项目一致） |

## 核心依赖

**零第三方依赖**（与原项目保持一致）

所有功能使用 Node.js 内置模块：
- `child_process` — 调用 rsync/ssh 命令
- `fs` / `path` — 文件操作
- `os` — 系统信息
- `http` / `https` — API 请求
- `stream` — 进度显示

## 外部工具依赖

| 工具 | 用途 | 必须 |
|------|------|------|
| `rsync` | 文件同步传输 | ✅ Linux/macOS 自带 |
| `ssh` | 远程连接 | ✅ 系统自带 |
| `node` | 远程运行 OpenClaw | ✅ |
| `npm` | 远程安装依赖 | ✅ |
